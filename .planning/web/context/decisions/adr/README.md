# Architecture Decision Records

Numbered, short-form decisions with permanent consequences. Each ADR captures: context (why this decision was needed), decision (what was chosen), consequences (what follows). Status of an ADR is one of `Accepted`, `Superseded by ADR-NNNN`, or `Rejected`.

For Phase 1 (§app2.1) the gate criteria require ADRs 1, 2, 3 to be filed.
